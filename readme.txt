A simple WordPress plugin to display a Coming Soon or Maintenance Mode page.

== Description ==

Nafeza Coming Soon plugin is a simple and user-friendly solution to let your website visitors know that your site is under construction or undergoing maintenance. You can customize the message displayed or set a specific page to act as your Coming Soon page. 

== Features ==

- Activate or Deactivate Coming Soon mode with one click.
- Customize the Coming Soon message.
- Option to set a specific page as the Coming Soon page.
- Translatable strings for localization.

== Installation ==

1. Upload the `nafeza-coming-soon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Settings' -> 'Coming Soon' to configure the plugin.

== Frequently Asked Questions ==

= Can I customize the Coming Soon page? =

Yes, you can either use a custom message or set a specific page as your Coming Soon page through the plugin's settings.

= Will it block access to my WordPress dashboard? =

No, logged-in users with the `edit_themes` capability (typically administrators and editors) will still have access to the dashboard and frontend.
